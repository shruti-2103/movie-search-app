# 🎬 Movie Search App

A simple and responsive **Movie Search Website** that allows users to search for movies and view details like poster, title, and release year using the OMDb API.

---

## 🚀 Live Demo

🌐 https://fav-movie-search-website-shruti.netlify.app/

---

## 📌 Features

* 🔎 Search any movie by name
* 🎬 Display movie posters and details
* 📱 Responsive design
* ⚡ Fast movie search using API

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* OMDb API
